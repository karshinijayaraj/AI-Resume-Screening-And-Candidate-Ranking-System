AI Resume Screening & Candidate Ranking System

An AI-powered recruitment application built with Python and Streamlit that automates resume screening, candidate ranking, and Gmail notifications.

Features

Upload multiple PDF resumes

Extract candidate names and email addresses

Match resumes against a job description

Rank candidates based on skill match percentage

Generate AI-based candidate summaries

Display an interactive recruitment dashboard

Export results as CSV

Send automated Gmail notifications to shortlisted candidates


Tech Stack

Python

Streamlit

Pandas

PyPDF2

SMTP (Gmail Automation)

Regular Expressions
